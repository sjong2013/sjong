One of the cool things about Angularjs is how it handles two-way binding. This means that it can bind elements to a Javascript variable and automatically update them whenever they change.
Type in the textbox below and see how the text below the box changes. This is because the text box is bound to a variable called "name".
Type a name in this box:
Name: {{name}}
